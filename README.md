# CodeAlpha Login & Registration System

## Description
A console-based Login and Registration System developed in C++ using File Handling.

## Features
- User Registration
- User Login
- Change Password
- Forgot Password
- Delete Account
- View Registered Users
- Total Registered Users

## Technologies Used
- C++
- File Handling (fstream)
- Visual Studio Code

## How to Run

Compile:
```bash
g++ main.cpp -o main
```

Run:
```bash
.\main.exe
```
